# 🍹 The Pour

A modern and responsive web application to explore, search, and create refreshing cocktail and mocktail recipes.

---

## 🚀 Live Demo

🔗 https://the-pour.netlify.app/

---

## ✨ Features

* 🍸 Browse cocktail & mocktail recipes
* 🔍 Search drinks by name
* 🧾 View ingredients and preparation steps
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* JavaScript / TypeScript
